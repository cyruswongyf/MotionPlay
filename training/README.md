# Training Scripts

This directory contains scripts for training custom gesture recognition models.

## Scripts

### `collect_gesture_data.py`

Collect gesture data for training custom classifier.

### `train_custom_gesture_classifier.py`

Train custom gesture classifier using collected data.

### `train_gesture_model.py`

Train gesture recognition model.

## Usage

1. First run `collect_gesture_data.py` to collect training data
2. Then run `train_custom_gesture_classifier.py` to train the model
3. Trained models will be saved in the `custom_gesture_recognizer/` directory

For more details, see the documentation files in the `docs/` directory.
